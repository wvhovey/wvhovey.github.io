Repetition

Riot Games: Valorant

valorant.com

Riot Games does an excellent job using repetition in their website for one of their games called Valorant. We can see there is repetition in their color scheme with the same pink color being used in many ways from buttons to subheadings to graphic elements. The darker blackish color is used in the main header as well as different shades and tints being used for subheadings and other text on the page.

Alignment

Santouka Hokkaido Ramen

santouka-usa.com

Santouka Hokkaido Ramen does fantastic work with their site, making sure every element is consistently aligned. In the menu view shown here, we can see how the images, heading, and description of each menu item are all aligned consistently to the left. In addition, each of the images are also aligned on their right side which makes this view very pleasing to the eye.

Proximity

Rotten Tomatoes

rottentomatoes.com

Rotten Tomatoes is a great example of proximity, keeping all related elements near each other and all unrelated elements farther away. There is proximity between the category headings and the pictures of various movie posters. This is how we know which movies belong to which category. In addition, the rating of each movie and its name are directly under the poster. This close proximity allows us to immediately recognize the relationship between the ratings and their respective movies.
